# PrimitivasGraficas_ComputacaoG

Entrada e Saída de dados:

Transformadas : 
A cada coordenada digitada aperte o botão OK.
Após inserir todas as coordenadas desejadas ,aperte o  botão FINALIZAR que então será apresentado as opções de trasformaçoes que podem ser feitas sobre os pontos digitados.

Primitivas Básicas :
Reta
Digite os pontos iniciais e finais da reta desejada, selecione o algoritmo que deseja utilizando os botoes GERAR DDA ,ou GERAR BRES.

Circunferencia :
Digite o centro e o raio, aperte o botão GERAR CIRC  ,que então a circunferência será exibida em nova tela.
Devido ao tamanho do frame de exibição dos resultados, as coordenadas digitadas, assim como o raio da circunferência possuem um limite de tamanho,portanto nem todos os valores digitados serão computados e será apresentado uma mensagem na tela alertando.
